# 🛒 Amazon Clone

## 📌 Overview  
This project is a front-end clone of the Amazon homepage built using HTML and CSS.  
It replicates the core layout and user interface of Amazon, focusing on structured design, navigation, and product display.


## 🔗 Live Demo  
👉 https://urvi1408.github.io/Amazon-Clone/


## ✨ Features  
- Navigation bar with logo, search bar, and cart  
- Product sections with cards and images  
- Hero banner section  
- Multi-column footer  
- Clean and structured layout similar to Amazon  


## 🛠️ Tech Stack  
- HTML5  
- CSS3 (Flexbox)


## 📂 Project Structure  
Amazon-Clone/
│── index.html
│── style.css
│── images/


## ⚙️ Run Locally  
Clone the repository and open `index.html` in your browser.


## 📚 Key Learnings  
- Building complex layouts inspired by real-world websites  
- Using Flexbox for alignment and spacing  
- Understanding UI structuring of e-commerce platforms  
- Replicating navigation and product sections
